# F-01: HTTP to HTTP 转发测试
# 需求: http 请求应能正确转发到 http 后端服务

# 规则: 将 http-forward.local 转发到本地 HTTP Echo Server
http-forward.local host://127.0.0.1:3000

# 测试用例:
# 1. 请求 http://http-forward.local/api/test 应该转发到 127.0.0.1:3000
# 2. 后端 Echo Server 应收到原始请求路径 /api/test
# 3. 后端应收到原始请求方法 GET
# 4. 响应应包含 Echo Server 返回的 JSON

# 带路径的转发
http-forward-path.local/api host://127.0.0.1:3000/backend
http-forward-xhost.local xhost://127.0.0.1:3000
