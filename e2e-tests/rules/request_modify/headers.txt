# R-01 ~ R-03: 请求头修改测试
# 需求: 代理应能添加、修改、删除请求头

# R-01: 添加请求头 (使用单个代码块变量)
req-header-add.local 127.0.0.1:3000 reqHeaders://{add_custom}
req-header-add.local reqHeaders://{add_reqid}

# R-02: 修改请求头 (覆盖 User-Agent)
req-header-modify.local 127.0.0.1:3000 reqHeaders://{modify_ua}

# R-03: 删除请求头
req-header-delete.local 127.0.0.1:3000 reqHeaders://{delete_encoding}

# 测试用例:
# 1. [添加] 请求后端应收到 X-Custom-Header: test-value
# 2. [修改] 请求后端应收到修改后的 User-Agent
# 3. [删除] 请求后端不应收到 Accept-Encoding 头

```add_custom
X-Custom-Header: test-value
```

```add_reqid
X-Request-Id: bifrost-test-123
```

```modify_ua
User-Agent: BifrostTest/1.0
```

```delete_encoding
Accept-Encoding:
```
