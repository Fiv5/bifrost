# C-01: Passthrough 规则测试
# 需求: passthrough 规则应使请求绕过后续规则并直接透传
# 兼容性: 旧的 ignore:// 写法会自动转换为 passthrough://

# 透传特定域名
ignore-this.local passthrough://

# 透传特定路径
*.local/health passthrough://
*.local/metrics passthrough://

# 提供一个可测试的正常转发规则
normal.local 127.0.0.1:3000

# 测试用例:
# 1. 请求 http://ignore-this.local/ 应直接透传，不经过后续规则处理
# 2. 请求任意域名的 /health 路径应直接透传
# 3. 透传请求不应有代理额外添加的响应头
