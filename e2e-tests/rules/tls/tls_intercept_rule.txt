# TLS 拦截规则测试
# 测试 tlsIntercept:// 和 tlsPassthrough:// 协议

# 强制解包 - 即使在白名单模式下未列入白名单，也会解包
*.force-intercept.test tlsIntercept://

# 强制不解包 - 即使在黑名单模式下，也直接透传
*.passthrough.test tlsPassthrough://

# 组合规则 - 解包后修改请求头
*.api.test tlsIntercept:// reqHeaders://(X-Intercepted: true)

# 组合规则 - 透传模式，不会修改请求
*.direct.test tlsPassthrough://
