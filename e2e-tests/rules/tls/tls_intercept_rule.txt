# TLS 拦截规则测试
# 测试 tlsIntercept:// 和 tlsPassthrough:// 协议
# 注意: 测试需要使用 HTTPS 协议才能真正验证 TLS 解包功能

# TLS-01: 强制解包 + 转发到 HTTPS mock 服务
# tlsIntercept:// 会拦截 TLS，允许后续规则修改请求
tls-intercept.test tlsIntercept:// https://127.0.0.1:3443

# TLS-02: 强制透传（不解包）
# tlsPassthrough:// 直接转发 TLS 流量，不修改
tls-passthrough.test tlsPassthrough://

# TLS-03: 解包后添加请求头并转发到 HTTPS
tls-add-header.test tlsIntercept:// https://127.0.0.1:3443 reqHeaders://`X-Intercepted:true`
