# P-04: IP地址优先于CIDR范围

# 精确IP匹配应该优先于CIDR范围匹配
# 例如: 对于 192.168.1.100，精确匹配 192.168.1.100 应优先于 192.168.1.0/24

# CIDR 范围规则 (低优先级)
192.168.1.0/24 resHeaders://{x-matched-by:cidr-range}

# 精确 IP 规则 (高优先级)
192.168.1.100 resHeaders://{x-matched-by:exact-ip}

# 测试用例说明:
# - 请求 192.168.1.100 时，应该命中精确IP规则
# - 请求 192.168.1.50 时，应该命中CIDR范围规则
