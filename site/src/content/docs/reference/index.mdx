## 细分能力

- 请求改写：`Headers`、`Body`、Cookies 与参数调整
- 响应改写：响应体、状态码、内容注入与代理结果调试
- URL 改写：域名、路径和查询参数的重写
- 流量控制：延迟、限速和节流
- 脚本规则：将动态逻辑接入到 Rules 体系
- WebSocket：面向长连接与消息流场景的支持
